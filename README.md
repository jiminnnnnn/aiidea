## ⚡핵심 기능
#### 1. 자연어 처리 기술 (NLP)를 활용하여 사용자의 음성을 인식하고 이해
#### 2. 음성 명령 사용자가 손쉽게 접근할 수 있도록 단순화된 디자인과 음성 피드백 제공\
#### 3. 사용자 주문 이력 및 선호도를 기반으로 개인화된 가게 추천 기능
#### 4. 마지막 단계에서의 인증 절차를 간소화
#### 5. 생체인식 또는 PIN입력 방식으로 간편하게 인증
